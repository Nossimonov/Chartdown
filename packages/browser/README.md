# @chartdown/browser

Browser embed for [Chartdown](https://github.com/Nossimonov/Chartdown) — one script tag renders every ` ```chartdown ` fenced block (or `<pre class="chartdown">`) on the page in place, entirely client-side.

```html
<script src="https://cdn.jsdelivr.net/npm/@chartdown/browser@0.1/dist/chartdown.browser.js" defer></script>
```

Write your map as plain text in the page; it becomes an SVG battlemap, hex chart, or region map on load. Try the [playground](https://nossimonov.github.io/Chartdown/) or see the [spec](https://github.com/Nossimonov/Chartdown/tree/main/docs/spec).
